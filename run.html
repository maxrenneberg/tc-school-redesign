Run the live school-settings tool

This loads the redesigned school-admin settings on top of trainchinese.com and reads/writes the account you're logged into. Design-only preview (no login needed): the demo.

Use a test account. It really saves. A few school-only features (remove teacher, permission toggles, give-subscription) only appear on an account that has multiple teachers and seats.

1. Add the bookmarklet

Drag this button up to your bookmarks bar (or right-click it → "Add to bookmarks"):

TC School Settings

…or create a bookmark manually and paste this as its URL / address:

javascript:(function(){var s=document.createElement('script');s.src='https://maxrenneberg.github.io/tc-school-redesign/live.js?t='+Date.now();document.body.appendChild(s);})();

2. Open it

  1. Log into a teacher or school account on trainchinese.com/v2.
  2. While on any /v2/ page, click the TC School Settings bookmark.
  3. The redesigned settings panel opens, wired to that account.

If the bookmark is fiddly

Open the browser dev console (F12) on a /v2/ page and paste this one line:
